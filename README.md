# DS402-Final-Project
How to run this code?

First thing the user wants to do is grab the Early stage diabetes risk prediction data set. Then, the user wants to import the data set using the import files function in Google Colab. Then the user wants to run the first code block in the code that imports all the required libraries and packages. Then, run all the data pre processing, cleaning, and exploratory data analysis code blocks to ensure clean and consistent results. After that you just follow all the code chunks in order. Everything is labeled!
